# FrostRise

Мобильное приложение для расчета глубины промерзания грунтов по методике СП 121.13330.2012 "Аэродромы"

## Описание приложения

FrostRise - это профессиональное мобильное приложение, разработанное для инженеров-строителей, проектировщиков и специалистов в области геотехники. Приложение позволяет проводить точные расчеты глубины промерзания грунтов с учетом различных параметров грунта, климатических условий и конструкционных слоев.

**Основное назначение:** Расчет и анализ промерзания грунтов для проектирования фундаментов, дорожных покрытий и других строительных конструкций.

## Функционал

### 📊 Основные возможности:
- **Расчет глубины промерзания** по методике СП 121.13330.2012
- **Анализ рисков** морозного пучения грунтов
- **Сравнение эффективности** теплоизоляционных материалов (Пеноплекс)
- **Многослойный расчет** с учетом различных конструкционных материалов

### 🏗️ Работа с данными:
- **База данных грунтов** с предустановленными параметрами ИГЭ
- **Климатические параметры** для различных регионов
- **Настраиваемые слои** конструкций (бетон, асфальт, щебень и др.)
- **Ручной ввод параметров** для нестандартных случаев

### 📈 Результаты и отчетность:
- **Детальные расчеты** с пошаговыми формулами
- **Визуализация результатов** в виде гистограмм
- **Генерация PDF-отчетов** с профессиональным оформлением
- **Оценка рисков** с цветовой индикацией

### 🎯 Дополнительные функции:
- Автоматическое сохранение расчетов
- История выполненных расчетов
- Экспорт данных в различные форматы
- Интуитивно понятный интерфейс на русском языке

### 🔧 Начало работы

#### Шаг 1: Выбор грунта
- Перейдите в раздел "Новый расчет"
- Выберите тип грунта из базы данных ИГЭ
- Или создайте пользовательский грунт с ручным вводом параметров

#### Шаг 2: Настройка климатических параметров
- Укажите среднюю температуру периода промерзания (°C)
- Задайте продолжительность промерзания (часы)
- Используйте предустановленные значения для вашего региона

#### Шаг 3: Добавление конструкционных слоев
- Нажмите "Добавить слой"
- Выберите тип материала (цементобетон, асфальт, щебень и др.)
- Укажите толщину слоя (м)
- Задайте параметры материала (плотность, влажность, теплопроводность)
- Повторите для всех необходимых слоев

#### Шаг 4: Выполнение расчета
- Нажмите кнопку "Рассчитать"
- Дождитесь обработки данных
- Просмотрите основные результаты на экране

### 📊 Анализ результатов

#### Основные показатели:
- **Глубина промерзания (Hₙ)** - расчетная глубина промерзания грунта
- **Высота промороженной толщи (Hf)** - общая высота промерзания
- **Оценка риска** - цветовая индикация уровня риска:
  - 🟢 Низкий риск
  - 🟡 Средний риск
  - 🔴 Высокий риск

#### Сравнение с Пеноплексом:
- Просмотрите эффективность теплоизоляции
- Анализируйте разницу в показателях Hf
- Оцените экономическую целесообразность применения утеплителя

### 📄 Генерация отчета

#### Создание PDF-документа:
1. На странице результатов нажмите "Создать PDF отчет"
2. Дождитесь генерации документа
3. Выберите действие:
   - "Поделиться" - отправить по email или в мессенджер
   - "Сохранить" - сохранить в память устройства

#### Структура отчета:
- Титульный лист с данными проекта
- Исходные данные расчета
- Расчетные слои конструкции
- Результаты расчета
- Детали расчета с формулами
